Lista nieuporządkowana

Jako ramy listy nieuporządkowanej używamy znacznika < ul > (unordered list).
Każdy element listy umieszczamy w znaczniku < li> (list item).

Przykład:

<p>Lista zakupów</p>

<ul>
   <li>Mleko</li>
   <li>Chleb</li>
  <li>Masło</li>
</ul>

Efekt:

Lista zakupów

  • Mleko
  • Chleb
  • Masło

Lista uporządkowana

Jako ramy listy uporządkowanej używamy znacznika < ol > (ordered list).
Każdy element listy umieszczamy w znaczniku < li> (list item).

Przykład:

<p>Lista zakupów</p>

<ol>
   <li>Mleko<li>
   <li>Chleb</li>
   <li>Masło</li>
</ol>

Efekt:

Lista zakupów

  1. Mleko
  2. Chleb
  3. Masło

Lista definicji

Jako ramy listy definicji używamy znacznika < dl > (definition list).
Termin definiowany umieszczamy w znaczniku < dt>, a definicję w znaczniku < dd>.

Przykład:

<dl>
   <dt>Krzesło</dt>
   <dd>Mebel do siedzenia.</dd>
   <dt>Długopis</dt>
   <dd>Przyrząd do pisania.</dd>
   <dt>Pies</dt>
   <dd>Zwierzę domowe.</dd>
</dl>

Efekt:

Lista definicji

Krzesło
Mebel do siedzenia.
Długopis
Przyrząd do pisania.
Pies
Zwierzę domowe.
