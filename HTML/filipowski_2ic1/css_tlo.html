Grafika jako tło

Aby obrazek po prostu powtarzał się na stronie (jest to domyślne), do znacznika body przypisujemy odpowiedni kod CSS:

body
	{
	background-image:url('../images/tlo_cegla.jpg');
	}

Efekt

Powtarzanie w pionie

Aby obrazek powtarzał się na stronie w pionie, dodajemy właściwość background-repeat :

body
	{
	background-image:url('../images/tlo_liscie.jpg');
	background-repeat:repeat-y;
	}

Efekt

Powtarzanie w poziomie

Aby obrazek powtarzał się na stronie w poziomie, dodajemy właściwość background-repeat :

body
	{
	background-image:url('images/tlo_liscie.jpg');
	background-repeat:repeat-x;
	}

Efekt

Stałe miejsce

Aby obrazek był w stałym miejscu, we właściwości background-repeat dajemy no-repeat.

Oto przykładowa strona z zastosowaniem tej właściwości:

<style>
	p	
		{
		width:800px;
		margin:0 auto;
		padding-top:600px;
		background: url('../images/tlo_lilia.jpg');
		background-repeat: no-repeat;
		background-color:#59A69E;
		}
</style>
<body>
	<p>Lorem ipsum...</p>
</body>

Efekt

Niezależny od przewijania

Aby obrazek był cały cały czas na swoim miejscu, niezależnie od przewijania strony używamy następujących właściwości background :

<style>
p	
	{
		width:800px;
		margin:0 auto;
		padding-top:50px;
		background: url('../images/tlo_smiley.gif');
		background-repeat: no-repeat;
		background-position: center;
		background-attachment:fixed;
	}
</style>
<body>
	<p>Lorem ipsum...</p>
</body>

Efekt (dla lepszego efektu można zmniejszyć okno przeglądarki)
